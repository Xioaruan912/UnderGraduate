# 直角坐标下的计算 

一般题目给出的 都是需要换序

也就是 先积分x 还是先积分y的讨论

```
后积先定限
限内画条线
先交写下限
后交写上限
```

并且对于二重积分的计算 我们一定要求上限>=下限 如果不是 一定要先 转化 这样才是我们确定的二重积分
